# How to run app

If you don't have a node version higher that v18.18.0 then u need to:

```bash
nvm install v18.18.0
```

and then: 

```bash
nvmw use v18.18.0
```

And right now you can install dependecies

```bash
npm i -f
```

And to run this app all you need to do right now is:

```bash
npm run dev
```
